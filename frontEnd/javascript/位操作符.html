# 按位取反 ~

按位取反~会将数值的32位二进制的每一位取反(0变为1，1变为0)。按位取反的操作符的本质取操作数的负值，然后减一。

// -24
~23

// -101
~100

// 9
～-10

// 100

# 小技巧：

  1. 向下取整
const num = 3.14;
console.log(~~num); // 3
1
2

按位取反~。将操作数取负值，并减1。对于浮点数，会直接舍弃小数的部分。比如~11.1 === -12。然后将结果，再次执行按位取反操作，就会得到了去除小数位的原数字，~-12 === 11。因此我们可以使用~~代替Math.floor。

  1. 判断索引是否存在
const name = "苏子龙";
if (~name.indexOf("苏")) {
	console.log("找到了")
} else {
	console.log("没找到")
}
// 找到了
