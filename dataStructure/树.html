# 树

# 什么是树？

树是一种分层数据的抽象模型。现实生活中最常见的树的例子是家谱，或是公司的组织架构图

# 树的相关术语

节点：树中的每一个元素都可以被称之为节点。

根节点：位于树顶部的节点叫作根节点（11）。它没有父节点。

内部节点：至少有一个子节点的节点称为内部节点。

外部节点：没有子元素的节点称为外部节点或叶节点。

子树：子树由节点和它的后代构成。例如，节点13、12和14构成了上图中树的一棵子树。

节点深度：节点的深度取决于它的祖先节点的数量。比如，节点3有3个祖先节点（5、7和11），它的深度为3。

树的高度：树的高度取决于所有节点深度的最大值。一棵树也可以被分解成层级。根节点在第0层，它的子节点在第1层，以此类推。上图中的树的高度为3（最大高度已在图中表示——第3层）。

# 二叉树

二叉树中的节点最多只能有两个子节点：一个是左侧子节点，另一个是右侧子节点。这个定义有助于我们写出更高效地在树中插入、查找和删除节点的算法。二叉树在计算机科学中的应用非常广泛。

# 二叉搜索树

二叉搜索树（BST）是二叉树的一种，但是只允许你在左侧节点存储（比父节点）小的值，在右侧节点存储（比父节点）大的值。

class Node {
  constructor(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}

class BinarySearchTree {
  constructor() {
    this.root = null;
  }

  insert(value) {
    if (this.root === null) {
      return (this.root = new Node(value));
    } else {
      return this.insertNode(this.root, value);
    }
  }

  insertNode(node, value) {
    if (node.value == value) return false;
    if (value < node.value) {
      if (node.left === null) {
        return (node.left = new Node(value));
      } else {
        return this.insertNode(node.left, value);
      }
    } else {
      if (node.right === null) {
        return (node.right = new Node(value));
      } else {
        return this.insertNode(node.right, value);
      }
    }
  }

  /**
   * 先序遍历（根结点->左子树->右子树）
   * @param {Function} cb
   * @param {Node} node
   */
  preorderTraversal(cb, node) {
    if (this.root === null || typeof cb !== "function") return;
    node = node || this.root;
    try {
      cb(node.value, node);
    } catch (error) {
      console.error(error);
    }
    node.left && this.preorderTraversal(cb, node.left);
    node.right && this.preorderTraversal(cb, node.right);
  }

  /**
   * 中序遍历（左子树->根结点->右子树）
   * @param {Function} cb
   * @param {Node} node
   */
  middleOrderTraversal(cb, node) {
    if (this.root === null || typeof cb !== "function") return;
    node = node || this.root;
    node.left && this.middleOrderTraversal(cb, node.left);
    try {
      cb(node.value, node);
    } catch (error) {
      console.error(error);
    }
    node.right && this.middleOrderTraversal(cb, node.right);
  }

  /**
   * 后序遍历（左子树->右子树->根结点）
   * @param {Function} cb
   * @param {Node} node
   */
  postorderTraversal(cb, node) {
    if (this.root === null || typeof cb !== "function") return;
    node = node || this.root;
    node.left && this.postorderTraversal(cb, node.left);
    node.right && this.postorderTraversal(cb, node.right);
    try {
      cb(node.value, node);
    } catch (error) {
      console.error(error);
    }
  }

  /**
   * 层次遍历
   * @param {*} cb
   */
  levelTraversal(cb) {
    if (this.root === null || typeof cb !== "function") return;
    const queue = [this.root];
    while (queue.length) {
      const node = queue.shift();
      if (node.left !== null) queue.push(node.left);
      if (node.right !== null) queue.push(node.right);
      try {
        cb(node.value, node);
      } catch (error) {
        console.error(error);
      }
    }
  }

  min(node) {
    node = node || this.root;
    let min = null;
    while (node) {
      min = node.value;
      node = node.left;
    }
    return min;
  }

  max(node) {
    node = node || this.root;
    let max = null;
    while (node) {
      max = node.value;
      node = node.right;
    }
    return max;
  }

  /**
   * 在树中查找一个键。如果节点存在，则返回true；如果不存在，则返回false。
   * @param {*} value
   */
  search(value) {
    let node = this.root;
    while (node) {
      if (node.value === value) return true;
      node = value < node.value ? node.left : node.right;
    }
    return false;
  }
}

const tree = new BinarySearchTree();
tree.insert(5);
tree.insert(3);
tree.insert(7);
tree.insert(2);
tree.insert(4);
tree.insert(6);
tree.insert(8);

console.log(tree.min()); // 2
console.log(tree.max()); // 8
console.log(tree.search(3)); // true
console.log(tree.search(8)); // true
console.log(tree.search(10)); // false

# 树的遍历

遍历一棵树是指访问树的每个节点并对它们进行某种操作的过程。但是我们应该怎么去做呢？应该从树的顶端还是底端开始呢？从左开始还是从右开始呢？访问树的所有节点有三种方式：中序、先序和后序。

# 中序遍历

中序遍历是一种以上行顺序访问BST所有节点的遍历方式，也就是以从最小到最大的顺序访问所有节点。中序遍历的一种应用就是对树进行排序操作。

# 先序遍历

先序遍历是以优先于后代节点的顺序访问每个节点的。先序遍历的一种应用是打印一个结构化的文档。

# 后序遍历

后序遍历则是先访问节点的后代节点，再访问节点本身。后序遍历的一种应用是计算一个目录及其子目录中所有文件所占空间的大小。
